+ 指數運算
+ Index Operations
+ ===================================
+ 
+ Level number：5
+ 
題型定義:
計算指數運算的結果

難度定義:
難度1: 簡單的指數運算，底數和指數均為個位數
難度2: 中等難度的指數運算，底數和指數為兩位數
難度3: 較複雜的指數運算，包含多個指數項的乘法
難度4: 複雜的指數運算，包含多個底數和指數的組合
難度5: 高難度指數運算，可能包含分數指數或負指數

題型設計:
1. 給定一個或多個指數運算表達式
2. 要求計算最終結果
3. 根據難度調整底數和指數的大小
4. 可能包含指數的乘法運算規則
5. 確保計算結果在合理範圍內

錯誤選項設計:
1. 錯誤使用指數運算法則（如：把指數相加而不是相乘）
2. 計算時忽略部分指數
3. 底數和指數位置顛倒
4. 運算順序錯誤

題目呈現:
計算以下指數運算的結果：
a^m × b^n = ?
（其中a、b為底數，m、n為指數）

答案要求:
- 結果應以最簡形式表示
- 如果結果過大，保持指數形式
- 使用 LaTeX 格式顯示（如 2^3）

備註:
- 難度1限制使用小的底數和指數
- 避免產生過大的數字
- 確保所有運算過程都在合理範圍內
- 可能需要應用指數運算的基本法則：
  * a^m × a^n = a^(m+n)
  * (a^m)^n = a^(m×n)
  * (a×b)^n = a^n × b^n
