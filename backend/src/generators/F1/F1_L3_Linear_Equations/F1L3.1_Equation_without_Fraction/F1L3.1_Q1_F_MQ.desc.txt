一元一次方程：基礎解法
Linear Equations: Basic Solving
===================================

分題數目：1 題
題型分佈：
Q1: 一元一次方程基礎解法 (難度: Fixed)

難度等級：
1. 基礎移項：ax + b = c 形式
   例：x + 5 = 2, x - 7 = 5
   特點：只有一個變量項，一個常數項等號右邊為數字

2. 標準形式：ax ± b = ±c
   例：-x + 8 = -4, -x + 4 = -8
   特點：變量項可能帶負號，等號右邊可能為負數

3. 同類項合併：ax + bx = c
   例：6x + 20 = 2x, 8x - 12 = 2x
   特點：等式兩邊都有變量項，需要合併同類項

4. 多項合併：ax ± b ± cx = d
   例：8z + 2 = z - 12, 5z + 8 = -6z - 14
   特點：包含多個變量項和常數項，需要多次移項和合併

5. 複雜係數：ax ± b = cx ± d（包含小數或較大數字）
   例：0.2x + 0.1 = 0.7, 2.3x - 1.7 = 2.9
   特點：包含小數係數或較大數字，計算較複雜

生成規則：
-----------------------------------
1. 變量規則：
   - 使用單一變量（x, y, z 等）
   - 係數範圍：整數 -12 到 12（不含0）
   - 小數係數（難度5）：限制到小數點後1位

2. 常數規則：
   - 整數範圍：-20 到 20
   - 小數（難度5）：限制到小數點後1位
   - 避免過大或過小的數字

3. 答案規則：
   - 難度1-4：盡量生成整數解
   - 難度5：可以有小數解（限制到小數點後1位）
   - 避免分數解

答案格式：
-----------------------------------
- 標準格式：x = n，其中n為最終答案
- 小數答案四捨五入到小數點後1位
- 需包含完整解題步驟說明

範例：
-----------------------------------
Q1 難度1：x + 5 = 2        解：x = -3
Q1 難度2：-x + 8 = -4      解：x = 12
Q1 難度3：6x + 20 = 2x     解：x = -5
Q1 難度4：8z + 2 = z - 12  解：z = -2
Q1 難度5：0.2x + 0.1 = 0.7 解：x = 3.0

注意事項：
-----------------------------------
1. 確保方程有唯一解
2. 避免生成係數為0的項
3. 避免生成解太大或太小的方程
4. 小數計算要注意精確度
5. 生成的方程要有教學意義，避免過於機械的題目
