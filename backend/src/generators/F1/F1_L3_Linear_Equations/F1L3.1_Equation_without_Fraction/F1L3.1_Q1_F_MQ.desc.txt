一元一次方程：基礎解法
Linear Equations: Basic Solving
===================================

Level number：5

難度等級：
1. 基礎移項：x ± b = c 形式
   例：x + 5 = 2, x - 7 = 5
   特點：變量係數為1，變量只在左邊，最多3項

2. 標準形式：ax ± b = c 形式
   例：-2x + 8 = -4, 3x - 4 = -8
   特點：變量係數可正可負，變量只在左邊，最多3項

3. 同類項合併：ax + bx = c 形式
   例：2x + 3x = 10, -2x + 5x = 6
   特點：兩個變量項，係數可正可負，最多3項

4. 多項合併：ax ± b ± cx = d 形式
   例：2x + 3 = -x + 8, 3x - 2 = -2x + 5
   特點：兩個變量項，係數可正可負，最多5項

5. 複雜係數：ax ± b = cx ± d（包含小數或較大數字）
   例：0.2x + 0.1 = 0.7, 2.3x - 1.7 = 2.9
   特點：包含小數係數或較大數字，計算較複雜

生成規則：
-----------------------------------
1. 變量規則：
   - 使用單一變量 x
   - Level 1：係數固定為1
   - Level 2-4：係數範圍：整數 -5 到 5（不含0）
   - Level 5：小數係數，限制到小數點後1位

2. 常數規則：
   - 整數範圍：-20 到 20
   - Level 1-2：變量只在等式左邊
   - Level 3-4：變量可在等式兩邊
   - Level 5：小數範圍 -10 到 10，限制到小數點後1位

3. 項數限制：
   - Level 1-3：最多3項
   - Level 4：最多5項
   - Level 5：最多4項

答案格式：
-----------------------------------
- 標準格式：x = n，其中n為最終答案
- Level 1-4：答案必須是整數
- Level 5：答案可以是小數（限制到小數點後1位）
- 需包含完整解題步驟說明

範例：
-----------------------------------
難度1：x + 5 = 2        解：x = -3
難度2：-2x + 8 = -4     解：x = 6
難度3：2x + 3x = 15     解：x = 3
難度4：2x + 3 = -x + 8  解：x = 2

注意事項：
-----------------------------------
1. 確保方程有唯一解
2. 避免生成係數為0的項
3. 避免生成解太大或太小的方程
4. 生成的方程要有教學意義
5. 確保每個難度等級的特點都被遵守
