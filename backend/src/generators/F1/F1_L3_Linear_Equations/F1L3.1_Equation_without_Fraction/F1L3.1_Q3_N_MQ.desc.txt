一元一次方程：多項式解法
Linear Equations: Polynomial Form
===================================

Level number：5

難度等級：
1. 基礎多項式：ax + b = cx + d 形式
   例：2 + 5(b + 3) = 3 + 2(4 + b)
   特點：每邊最多兩項，簡單括號運算

2. 負係數多項式：-ax - b = -cx - d 形式
   例：14 - 3(5 - c) = 4 - (c + 1)
   特點：包含負係數，簡單括號運算

3. 雙括號多項式：a(x + b) + c(x + d) = e(x + f) 形式
   例：2(3z + 1) + 3(z - 2) = 4(z + 7) - 9(z + 2)
   特點：多個括號，需要分配律展開

4. 複雜係數多項式：a(bx + c) + d(ex + f) = g(hx + i) 形式
   例：4(2z + 1) - 7(3 - z) = 5(z + 3) - 7(5 - z)
   特點：括號內有係數，計算較複雜

5. 高階多項式：多個不同形式的括號組合
   例：7(a + 3) - 2(2 - a) = 5(3a - 1) + 22(a + 1)
   特點：多個不同類型的括號，需要仔細整理同類項

生成規則：
-----------------------------------
1. 係數規則：
   - Level 1：係數範圍 1-5
   - Level 2-3：係數範圍 -9 到 9（不含0）
   - Level 4-5：係數範圍 -12 到 12（不含0）

2. 常數規則：
   - Level 1-2：常數範圍 -10 到 10
   - Level 3-4：常數範圍 -15 到 15
   - Level 5：常數範圍 -20 到 20

3. 括號數量：
   - Level 1：每邊 1 個括號
   - Level 2：至少一邊有括號
   - Level 3：每邊 1-2 個括號
   - Level 4：每邊 2-3 個括號
   - Level 5：每邊最多 3 個括號

4. 特殊要求：
   - 避免生成係數為0的項
   - 確保方程有唯一解
   - 避免分母為0的情況
   - 避免過於複雜的計算

答案格式：
-----------------------------------
- 標準格式：x = n，其中n為最終答案
- 答案必須是整數
- 答案範圍限制在 -10 到 10 之間
- 需包含完整解題步驟說明

範例：
-----------------------------------
難度1：2 + 5(b + 3) = 3 + 2(4 + b)        解：b = -1
難度2：14 - 3(5 - c) = 4 - (c + 1)        解：c = 2
難度3：2(3z + 1) + 3(z - 2) = 4(z + 7)    解：z = -3
難度4：4(2z + 1) - 7(3 - z) = 5(z + 3)    解：z = 4
難度5：7(a + 3) - 2(2 - a) = 5(3a - 1)    解：a = -2

注意事項：
-----------------------------------
1. 確保方程有唯一解
2. 避免生成係數為0的項
3. 避免生成解太大或太小的方程
4. 生成的方程要有教學意義
5. 確保每個難度等級的特點都被遵守
