一元一次方程：多項式解法
Linear Equations: Polynomial Form
===================================

Level number：3

難度等級：
1. 基礎多項式：c1 + a1(x + b1) = c2 + a2(x + b2) 形式
   例：3 + 3(x + 3) = -18 + 4(x - 2)
   特點：每邊一個括號，係數為正數，簡單括號運算

2. 負係數多項式：c1 + a1(b1 - x) = c2 - (x + b2) 形式
   例：14 - 3(5 - x) = 4 - (x + 1)
   特點：包含負係數，需要注意正負號變化

3. 雙括號多項式：a1(b1x + c1) + a2(x + c2) = a3(x + c3) + a4(x + c4) 形式
   例：4(5x - 7) + 5(x + 5) = -1(x + 14) + 6(x + 28)
   特點：每邊兩個括號，需要分配律展開

生成規則：
-----------------------------------
1. 係數規則：
   - Level 1：係數範圍 1-5（正數）
   - Level 2：係數範圍 -9 到 -1（負數）
   - Level 3：係數範圍 -9 到 9（不含0）

2. 常數規則：
   - Level 1：常數範圍 -5 到 5
   - Level 2：常數範圍 -9 到 9
   - Level 3：常數範圍 -15 到 15

3. 括號數量：
   - Level 1：每邊 1 個括號，形式固定
   - Level 2：每邊 1 個括號，包含減號
   - Level 3：每邊 2 個括號，形式較複雜

4. 特殊要求：
   - 確保所有常數都是整數
   - 確保方程有唯一解
   - 解的範圍限制在 -10 到 10 之間
   - 避免係數為0的情況

答案格式：
-----------------------------------
- 標準格式：x = n，其中n為整數解
- 答案範圍限制在 -10 到 10 之間
- 需包含完整解題步驟說明：
  1. 展開括號
  2. 移項
  3. 合併同類項
  4. 求解

範例：
-----------------------------------
難度1：3 + 3(x + 3) = -18 + 4(x - 2)        解：x = 4
難度2：14 - 3(5 - x) = 4 - (x + 1)          解：x = 2
難度3：4(5x - 7) + 5(x + 5) = -1(x + 14) + 6(x + 28)   解：x = 6

注意事項：
-----------------------------------
1. 確保方程有唯一解
2. 避免生成係數為0的項
3. 避免分數出現在常數項中
4. 生成的方程要有教學意義
5. 確保每個難度等級的特點都被遵守
