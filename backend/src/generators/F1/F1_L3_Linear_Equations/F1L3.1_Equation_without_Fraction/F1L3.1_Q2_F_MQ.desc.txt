一元一次方程：括號解法
Linear Equations: Solving with Brackets
===================================

Level number：5

難度等級：
1. 基礎括號：a(x + b) = c 形式
   例：4(x + 1) = 16, 2(x - 3) = 10
   特點：一個括號，係數為正整數，括號內最多2項

2. 負係數括號：-a(x + b) = c 形式
   例：-3(x + 2) = -12, -2(x - 4) = 8
   特點：一個括號，係數為負整數，括號內最多2項

3. 雙括號：a(x + b) = c(x + d) 形式
   例：2(x + 3) = 3(x + 1)
   特點：兩個括號，係數為整數，括號內最多2項

4. 複雜括號：a(bx + c) = d 形式
   例：2(3x - 4) = 10, 3(2x + 5) = 24
   特點：括號內的變量有係數，計算較複雜

5. 混合運算：a(bx + c) = d(ex + f) 形式
   例：2(3x - 1) = 3(2x + 4)
   特點：兩個括號，括號內的變量都有係數

生成規則：
-----------------------------------
1. 係數規則：
   - Level 1：a 為 1-5 的正整數
   - Level 2：a 為 -5 到 -1 的負整數
   - Level 3-5：係數範圍：整數 -5 到 5（不含0）

2. 括號內常數規則：
   - Level 1-2：範圍 -5 到 5
   - Level 3-5：範圍 -10 到 10

3. 答案限制：
   - 所有難度等級答案必須是整數
   - 答案範圍限制在 -10 到 10 之間

4. 特殊要求：
   - 避免生成係數為0的項
   - 確保方程有唯一解
   - 避免分母為0的情況
   - 避免過於複雜的計算

答案格式：
-----------------------------------
- 標準格式：x = n，其中n為最終答案
- 答案必須是整數
- 需包含完整解題步驟說明

範例：
-----------------------------------
難度1：4(x + 1) = 16       解：x = 3
難度2：-2(x - 4) = -8      解：x = 0
難度3：2(x + 3) = 3(x + 1) 解：x = -5
難度4：3(2x - 1) = 15      解：x = 3
難度5：2(3x - 1) = 4(x + 2) 解：x = 2

注意事項：
-----------------------------------
1. 確保方程有唯一解
2. 避免生成係數為0的項
3. 避免生成解太大或太小的方程
4. 生成的方程要有教學意義
5. 確保每個難度等級的特點都被遵守
