代數式運算：基礎運算
Basic Algebraic Operations
===================================

分題數目：1 題
題型分佈：
Q1: 基礎代數式運算 (難度: Fixed)

難度等級：
1. 單項式與單項式相加減
2. 單項式與多項式相加減
3. 多項式與多項式相加減
4. 含括號的多項式運算
5. 複雜的多項式運算（含括號嵌套）

生成規則：
-----------------------------------
1. 變量規則：
   - 使用單一變量 x
   - 指數範圍：1-2
   - 係數範圍：-10 到 10（不含0）

2. 項數規則：
   - 每個運算式包含2-4個項
   - 至少包含一個變量項
   - 可包含常數項

3. 運算規則：
   - 只涉及加減運算
   - 可能包含括號
   - 避免重複的項

答案格式：
-----------------------------------
- 合併同類項
- 按指數降冪排列
- 係數化簡至最簡形式
- 省略係數為1的情況（除非是常數項）
- 省略指數為1的情況

範例：
-----------------------------------
Q1 難度1：2x + 3x = 5x
Q1 難度2：(3x + 2) + (2x - 1) = 5x + 1
Q1 難度3：(2x² + 3x) + (-x² + 2x) = x² + 5x
Q1 難度4：2(x + 3) - (2x - 1) = -x + 7
Q1 難度5：3(x² - 2) - (2x² + 1) = x² - 7

注意事項：
-----------------------------------
1. 確保生成的題目係數簡單，易於計算
2. 避免生成係數為0的項
3. 確保最終答案不會消去所有變量項
4. 括號的使用要合理，避免不必要的括號
5. 答案必須是最簡形式
