在坐標平面上找出點的象限
Find the Quadrant of a Point in the Coordinate Plane
===================================

Level number：5

難度等級：
1. 基礎坐標：四個象限的整數點
   例：點 A(-2, -3) 在哪一個象限？
   特點：點可能出現在任何象限，坐標都是整數

2. 擴展坐標：包含坐標軸,原點上的點
   例：點 A(0, -2) 在哪一個位置？
   特點：點可能在坐標軸上，需要分辨是否屬於象限

3. 多點判斷：四個點在不同象限
   例：哪一個點在第二象限？
   特點：四個點分別在不同象限，需要判斷指定象限的點

4. 多點特殊位置：點在坐標軸或原點
    例：哪一個點在 x 軸上？
    特點：四個點分別在 x 軸上、y 軸上、原點和某個象限
    點的位置：
    - 一個點在 x 軸上
    - 一個點在 y 軸上
    - 一個點在原點
    - 一個點在任意象限

5. 坐標軸正負方向：判斷點在坐標軸的正負方向
    例：哪一個點的 x 坐標為正數？或哪一個點的 y 坐標為負數？
    特點：四個點分別在：
    - 如果問 x 坐標為正數，則只有一個點的 x > 0，其他點的 x ≤ 0
    - 如果問 x 坐標為負數，則只有一個點的 x < 0，其他點的 x ≥ 0
    - 如果問 y 坐標為正數，則只有一個點的 y > 0，其他點的 y ≤ 0
    - 如果問 y 坐標為負數，則只有一個點的 y < 0，其他點的 y ≥ 0

生成規則：
-----------------------------------
1. 坐標生成：
   - 整數坐標範圍：-5 到 5
   - 確保點不會太接近坐標軸（除非刻意測試）
   - 難度5時，確保只有一個點滿足條件（正/負坐標）

2. 選項生成：
   難度1：
   - 在坐標平面上標示一個點，問該點在哪一個象限
   - 點的坐標為整數，不在坐標軸上
   
   難度2：
   - 在坐標平面上標示一個點，問該點在哪一個位置
   - 點可能在坐標軸上或原點
   
   難度3：
   - 在坐標平面上標示四個點 P、Q、R、S，每個點在不同象限
   - 問哪一個點在指定象限
   
   難度4：
   - 在坐標平面上標示四個點 P、Q、R、S
   - 三個點分別在 x 軸上、y 軸上和原點
   - 一個點在任意象限
   - 問哪一個點在指定位置（x 軸上、y 軸上或原點）

   難度5：
   - 在坐標平面上標示四個點 P、Q、R、S
   - 根據問題類型（x/y坐標的正/負），確保只有一個點滿足條件
   - 其他三個點都不滿足條件
   - 問哪一個點的坐標滿足指定條件（正/負）

答案格式：
-----------------------------------
- 第一象限：回答「第一象限」
- 第二象限：回答「第二象限」
- 第三象限：回答「第三象限」
- 第四象限：回答「第四象限」
- 特殊位置：
  * x 軸上：回答「在 x 軸上」
  * y 軸上：回答「在 y 軸上」
  * 原點：回答「在原點」

難度5：
- 答案形式：選擇滿足條件的點的標籤（P、Q、R、S）

範例：
-----------------------------------
難度1：點 A(-1, -2) 在哪一個象限？ 解：第三象限
難度2：點 A(0, -3) 在哪一個位置？ 解：在 y 軸上
難度3：點 A(0, 0) 在哪一個位置？ 解：在原點
難度4：點 A(3, 4) 在哪一個象限？ 解：第一象限

難度5：
問：哪一個點的 x 坐標為正數？
- 點 P(3, 2)：x > 0，是正確答案
- 點 Q(-1, 4)：x < 0
- 點 R(0, -2)：x = 0
- 點 S(-2, 1)：x < 0

注意事項：
-----------------------------------
1. 坐標平面顯示要求：
   - 清晰標示 x 軸和 y 軸
   - 顯示適當的格線
   - 標示主要刻度

2. 教學重點：
   - 象限的定義是由坐標的正負決定
   - 坐標軸上的點不屬於任何象限
   - 原點不屬於任何象限也不屬於坐標軸

3. 答案解釋：
   - 說明如何通過觀察 x、y 坐標的正負判斷象限
   - 對於坐標軸上的點，特別說明其位置特徵
   - 使用顏色標示來突出坐標的正負性

4. 常見錯誤提醒：
   - 混淆象限的順序
   - 誤把坐標軸上的點歸入象限
   - 忽視原點的特殊性

5. 難度5特別注意：
   - 強調坐標軸的正負方向概念
   - 說明正/負方向與坐標值的關係
   - 使用不同顏色標示正負方向
   - 確保學生理解坐標值與位置的對應關係
