因式分解 (交叉相乘法 - 雙變量)
Factorization (Cross Method - Two Variables)
===================================

Level number：5

題型：將二次項分解因式（二次項係數為1，包含兩個變量）
Factorize quadratic expressions with two variables where coefficient of x² is 1

生成邏輯：
- 先生成兩個整數因子 m, n（注意：m, n 不能為 0）
- 題目形式為：x² + bxy + cy²，其中：
  * b = -(m + n)  [xy項係數]
  * c = m × n     [y²項係數]
- 答案形式為：(x + my)(x + ny)

特殊限制：
1. m, n 不能為 0（避免線性因式）
2. m ≠ -n（避免xy項係數為 0）
3. m ≠ n（避免重複因式）

難度等級：
1. 簡單：全為負整數
   - m 範圍：-1 到 -4，不含 0
   - n 範圍：-1 到 -9，不含 0，且 |n| > |m|
   - 生成題目形式：x² + bxy + cy²，其中 b, c 均為正數
   - 例如：x² + 5xy + 6y² = (x + 2y)(x + 3y)

2. 中等：全為正整數
   - m 範圍：1 到 4，不含 0
   - n 範圍：1 到 9，不含 0，且 n > m
   - 生成題目形式：x² - bxy + cy²，其中 b, c 均為正數
   - 例如：x² - 7xy + 12y² = (x - 3y)(x - 4y)

3. 較難：一正一負
   - m 範圍：-8 到 -2，不含 0
   - n 範圍：2 到 8，不含 0，且 n ≠ -m
   - 生成題目形式：x² + bxy - cy²，其中 b, c 的絕對值較大
   - 例如：x² + 3xy - 10y² = (x + 2y)(x - 5y)

4. 進階：其他變量組合
   - 使用其他變量組合，如：a² + bap + cp²
   - m, n 範圍：-7 到 17，不含 0
   - 變量選擇：從 [x,y,z,a,b,p,q,r,s,t] 中隨機選擇兩個
   - 例如：p² - 5pq + 6q² = (p - 2q)(p - 3q)

5. 挑戰：完全平方式
   - m 範圍：2 到 13，不含 0
   - 生成題目形式（隨機選擇以下三種類型之一）：
     a. 完全平方差：x² - m²y² = (x + my)(x - my)
        例如：x² - 16y² = (x + 4y)(x - 4y)
     b. 完全平方式：x² + 2mxy + m²y² = (x + my)²
        例如：x² + 6xy + 9y² = (x + 3y)²
     c. 完全平方式：x² - 2mxy + m²y² = (x - my)²
        例如：x² - 8xy + 16y² = (x - 4y)²

錯誤選項生成策略：
1. 符號錯誤：
   - 將 (x + my)(x + ny) 中的符號反轉
   - 例如：(x - 2y)(x - 3y) 寫成 (x + 2y)(x + 3y)

2. 數值錯誤：
   - 將係數加減1
   - 隨機改變一個係數

3. 展開錯誤：
   - 忽略負號
   - 係數計算錯誤
   - 確保錯誤答案仍是合理的因式形式

難度5的特殊錯誤選項：
1. 完全平方差類型時：
   - 使用相鄰的平方數
   - 使用 (x + my)² 或 (x - my)²
   - 使用不同的係數

2. 完全平方式類型時：
   - 符號錯誤：(x - my)² 而不是 (x + my)²，反之亦然
   - 係數錯誤：使用相鄰的數
   - 展開錯誤：x² ± mxy + m²y² 而不是 x² ± 2mxy + m²y²

注意事項：
1. 確保生成3個不同的錯誤選項
2. 錯誤選項應該看起來合理
3. 避免重複的錯誤選項
4. 確保錯誤選項展開後與原式不同
5. 所有因子都不能為0
6. 避免xy項係數為0的情況（除了難度5的完全平方差）
7. 避免重複因式的情況
