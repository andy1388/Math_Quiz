因式分解 (交叉相乘法)
Factorization (Cross Method)
===================================

題型：將二次項分解因式（二次項係數為1）
Factorize quadratic expressions where coefficient of x² is 1

生成邏輯：
- 先生成兩個整數因子 m, n（注意：m, n 不能為 0）
- 題目形式為：x² + bx + c，其中：
  * b = -(m + n)  [一次項係數]
  * c = m × n     [常數項]
- 答案形式為：(x - m)(x - n)

特殊限制：
1. m, n 不能為 0（避免線性因式）
2. m ≠ -n（避免一次項係數為 0）
3. m ≠ n（避免重複因式）

難度等級：
1. 簡單：全為負整數
   - m 範圍：-1 到 -4，不含 0
   - n 範圍：-1 到 -9，不含 0，且 |n| > |m|
   - 生成題目形式：x² + bx + c，其中 b, c 均為正數
   - 例如：x² + 5x + 6 = (x + 2)(x + 3)

2. 中等：全為正整數
   - m 範圍：1 到 4，不含 0
   - n 範圍：1 到 9，不含 0，且 n > m
   - 生成題目形式：x² - bx + c，其中 b, c 均為正數
   - 例如：x² - 7x + 12 = (x - 3)(x - 4)

3. 較難：一正一負
   - m 範圍：-8 到 -2，不含 0
   - n 範圍：2 到 8，不含 0，且 n ≠ -m
   - 生成題目形式：x² + bx - c，其中 b, c 的絕對值較大
   - 例如：x² + 3x - 10 = (x + 2)(x - 5)

4. 進階：混合範圍
   - m, n 範圍：-7 到 17，不含 0，且 m ≠ n，m ≠ -n
   - 生成題目形式：x² ± bx ± c，係數變化較大
   - 例如：x² + 2x - 15 = (x + 3)(x - 5)

5. 挑戰：完全平方式
   - m 範圍：2 到 8，不含 0
   - 生成題目形式（隨機選擇以下三種類型之一）：
     a. 完全平方差：x² - m²  = (x + m)(x - m)
        例如：x² - 16 = (x + 4)(x - 4)
     b. 完全平方式：x² + 2mx + m²  = (x + m)²
        例如：x² + 6x + 9 = (x + 3)²
     c. 完全平方式：x² - 2mx + m²  = (x - m)²
        例如：x² - 8x + 16 = (x - 4)²

難度5的特殊錯誤選項：
1. 完全平方差類型時：
   - 使用相鄰的平方數，如 (x + 3)(x - 3) 而不是 (x + 4)(x - 4)
   - 使用 (x + m)(x + m) 或 (x - m)(x - m)
   - 使用不同的係數，如 (x + m)(x - n) 其中 n ≠ m

2. 完全平方式類型時：
   - 符號錯誤：(x - m)² 而不是 (x + m)²，反之亦然
   - 係數錯誤：使用相鄰的數，如 (x + 3)² 而不是 (x + 4)²
   - 展開錯誤：x² ± mx + m² 而不是 x² ± 2mx + m²

解題步驟說明：
1. 觀察二次項的係數：
   - 完全平方差：x² - m²
   - 完全平方式：x² ± 2mx + m²

2. 辨認題型：
   a. 若常數項為負的平方數，且無一次項，使用完全平方差公式
   b. 若一次項係數為常數項平方根的2倍，使用完全平方式
   c. 若一次項係數為常數項平方根的-2倍，使用完全平方式

3. 寫出因式分解形式：
   a. 完全平方差：a² - b² = (a + b)(a - b)
   b. 完全平方式：x² + 2mx + m² = (x + m)²
   c. 完全平方式：x² - 2mx + m² = (x - m)²

注意事項：
1. 確保生成3個不同的錯誤選項
2. 錯誤選項應該看起來合理
3. 避免重複的錯誤選項
4. 確保錯誤選項展開後與原式不同
5. 所有因子都不能為0
6. 避免一次項係數為0的情況（除了難度5）
7. 避免重複因式的情況

錯誤選項生成策略：
1. 符號錯誤：
   - 將 (x ± m)(x ± n) 中的符號反轉
   - 例如：(x - 2)(x - 3) 寫成 (x + 2)(x + 3)

2. 數值錯誤：
   - 交換 m 和 n 的位置
   - 將係數加減1
   - 隨機改變一個係數

3. 展開錯誤：
   - 忽略負號
   - 係數計算錯誤
   - 確保錯誤答案仍是合理的因式形式

難度5的特殊錯誤選項：
1. 錯誤的平方數：
   - 使用相鄰的平方數，如 (x + 3)(x - 3) 而不是 (x + 4)(x - 4)
2. 符號錯誤：
   - 使用 (x + m)(x + m) 或 (x - m)(x - m)
3. 數值錯誤：
   - 使用不同的係數，如 (x + m)(x - n) 其中 n ≠ m

注意事項：
1. 確保生成3個不同的錯誤選項
2. 錯誤選項應該看起來合理
3. 避免重複的錯誤選項
4. 確保錯誤選項展開後與原式不同
5. 所有因子都不能為0
6. 避免一次項係數為0的情況（除了難度5）
7. 避免重複因式的情況

解題步驟說明：
1. 觀察二次項的係數：
   - 一次項係數 b 是兩個因子之和的相反數
   - 常數項 c 是兩個因子的積

2. 找出兩個因子 m, n：
   - m + n = -b
   - m × n = c

3. 寫出因式分解形式：
   - (x - m)(x - n)，其中按規則排序
   - 難度5使用完全平方差公式：a² - b² = (a + b)(a - b)
