計算以下常用對數 (使用計算機)
Calculate the following common logarithm expressions using calculator
===================================

Level number: 7

題型：計算進階常用對數值（以10為底）
Question Type: Calculate advanced common logarithm values (base 10)

難度等級：
-----------------------------------
1. 10的整數次方
   例：log 10 = 1、log 1000 = 3
   特點：結果為整數

2. 小數點後多位
   例：log 0.01 = -2、log 0.00001 = -5
   特點：結果為負整數

3. 數字運算
   例：log(2×10×500)、log(4×5000×0.05)
   特點：需要先計算括號內的數值

4. 10的正負次方
   例：log 10^500、log 10^(-200)
   特點：結果為正負整數

5. 根號運算
   例：log √1000、log ∜100000
   特點：需要先計算根號

6. 分數運算
   例：log (1/√10)、log (1/∜100)
   特點：需要先計算分數

7. 指數形式的對數
   例：10^(log 2)、10^(log x)
   特點：需要理解對數的指數性質
   說明：這類題目需要使用對數的性質來解答
   - 10^(log x) = x
   - 10^(log 2) = 2

生成規則：
-----------------------------------
1. 數字範圍：
   - 10的次方：-1000 到 1000
   - 根號內數字：1 到 100000
   - 乘法運算：結果在 0.00001 到 100000 之間
   - 分數：分子為1，分母為根號形式
   - 指數形式：使用簡單的數字如2、3、5等

2. 特殊要求：
   - 所有答案需準確到小數點後3位
   - 避免生成過於複雜的計算
   - 確保計算器能夠處理所有生成的數值

答案格式：
-----------------------------------
小數（準確到3位）
例：
- log 10 = 1.000
- log 0.01 = -2.000
- log(2×10×500) = 4.000
- log 10^500 = 500.000
- log √1000 = 1.500
- log (1/√10) = -0.500

錯誤答案生成規則：
-----------------------------------
1. 正負號錯誤：
   - 忽略負號
   - 例：log 0.01 誤算為 2.000（正確為 -2.000）

2. 運算順序錯誤：
   - 未先計算括號內的數值
   - 例：log(2×10×500) 誤算為其他值

3. 根號處理錯誤：
   - 未正確處理根號的次方
   - 例：log √1000 誤算為 3.000（正確為 1.500）

4. 指數形式錯誤：
   - 未正確理解10^(log x) = x的性質
   - 例：10^(log 2) 誤算為 log 2 或 10^2

解題步驟範例：
-----------------------------------
計算 log(2×10×500)：

1. 先計算括號內的數值
   2×10×500 = 10000

2. 計算 log 10000
   = 4.000

[使用計算機計算，答案準確到小數點後3位]

計算 10^(log 2)：

1. 理解題目形式
   10^(log 2) 是將log 2作為指數

2. 應用對數性質
   10^(log x) = x
   所以 10^(log 2) = 2

[這類題目主要考察對數的性質理解]
