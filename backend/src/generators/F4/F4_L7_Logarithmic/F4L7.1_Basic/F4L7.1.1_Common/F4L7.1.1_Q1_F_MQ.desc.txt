計算以下常用對數 (使用計算機)
Calculate the following common logarithm expressions using calculator
===================================

Level number：4

題型：計算常用對數值（以10為底）
Question Type: Calculate common logarithm values (base 10)

難度等級：
-----------------------------------
1. 整數 2-9
   例：log 2 = 0.301、log 5 = 0.699
   特點：結果為小於1的正數

2. 數字1
   例：log 1 = 0
   特點：結果為0

   特殊數字：
   - log 10 = 1
   - log 0 = Undefined
   - log(-1) = Undefined

3. 較大整數 10-9999
   例：log 100 = 2.000、log 5000 = 3.699
   特點：結果為大於1的正數

4. 小數 0.1-0.9
   例：log 0.5 = -0.301
   特點：小於1的數得到負數結果

生成規則：
-----------------------------------
1. 數字範圍：
   - 整數：2-9
   - 數字1
   - 較大整數：10-9999
   - 小數：0.1-19.9

2. 特殊要求：
   - 所有答案需準確到小數點後3位
   - 避免生成過於接近的數值
   - 確保計算器能夠處理所有生成的數值

答案格式：
-----------------------------------
小數（準確到3位）
例：
- log 2 = 0.301
- log 1 = 0.000
- log 100 = 2.000
- log 0.5 = -0.301

錯誤答案生成規則：
-----------------------------------
1. 正負號錯誤：
   - 忽略負號
   - 例：log 0.5 誤算為 0.301（正確為 -0.301）

2. 小數點位置錯誤：
   - 向左或向右移動一位
   - 例：log 100 誤算為 0.200（正確為 2.000）

3. 計算錯誤：
   - 數值略微偏差（±0.1以內）
   - 例：log 2 誤算為 0.315（正確為 0.301）

-----------------------------------

log 0.5
