計算以下常用對數 (不使用計算機)
Evaluate the following common logarithm expressions without using calculator
===================================

題型：計算常用對數值（以10為底）
Question Type: Calculate common logarithm values (base 10)

生成邏輯：
- 生成不同類型的常用對數表達式
- 答案需要通過對數運算法則求解
- 確保答案為整數

題目範圍：
1) 簡單整數對數 (Level 1)
   - log(10^n), n的範圍：1-5
   例如：
   - log(10^1) = 1
   - log(10^3) = 3
   - log(10^5) = 5

2) 小數和分數對數 (Level 2)
   - log(10^n), n的範圍：-1至-6
   - 隨機使用小數或分數形式
   例如：
   - log(0.1) = log(1/10) = -1
   - log(0.001) = log(1/1000) = -3
   - log(0.000001) = log(1/1000000) = -6

3) 乘積與商的對數 (Level 3)
   - log(a×b×c)，其中：
     * a, b, c 均為整數
     * a×b×c 的結果必須是10的整數次方
   例如：
   - log(2×5×100) = log(1000) = 3
   - log(4×25×10) = log(1000) = 3
   - log(2×2×250) = log(1000) = 3

4) 根式對數 (Level 4)
   - log(√x)
   - log(∛x)
   例如：
   - log√1000
   - log∜100000

注意事項：
1. 確保所有數值都可以通過對數運算法則求解
2. 避免需要計算機才能解決的題目
3. 答案應該是整數或簡單分數
4. 題目難度要循序漸進

適用年級：
高中數學第四冊
