Evaluate the following common logarithm expressions using calculator
===================================

題型：計算常用對數值（以10為底）
Question Type: Calculate common logarithm values (base 10)

生成邏輯：
- 生成不同類型的常用對數表達式
- 答案需要準確到小數點後3位

題目範圍：
- log 2 到 log 9 (整數)
- log 1
- log 10 到 log 9999 (較大整數)
- log 0.1 到 log 0.9 (小數)
例如：
- log 2 = 0.301
- log 5.4 = 0.732
- log 0.54 = -0.268
- log 1 = 0
- log 2000 = 3.301

答案呈現：

- 負數答案需要明確顯示負號

   - 小數點位置偏移
   - 捨入錯誤

3. 計算錯誤
   - 使用錯誤的對數運算法則
   - 常見的計算錯誤
