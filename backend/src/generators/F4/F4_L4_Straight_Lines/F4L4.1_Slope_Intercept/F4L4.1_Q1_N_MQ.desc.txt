題型定義：
求直線方程的斜率截距式 y = mx + c

難度定義：
【給定點和斜率求方程】
難度1: 給定整數坐標點和正整數斜率
- 坐標範圍：(-5, 5)
- 斜率範圍：1 到 5 的正整數
- 截距要求：正整數且不超過10

難度2: 給定整數坐標點和簡單分數斜率
- 坐標範圍：(-5, 5)
- 斜率形式：分母為2,3,4,5的分數
- 分子範圍：(-5, 5)的非零整數
- 截距要求：整數且絕對值不超過10

難度3: 給定整數坐標點和斜率（可能是整數或分數）
- 坐標範圍：(-5, 5)
- 斜率形式：50%機率為整數（-5到5，不含0），50%機率為分數（分母2-5）
- 截距要求：整數或簡單分數

【給定兩點求方程】
難度4: 給定兩個整數坐標點，求通過這兩點的直線方程
- 坐標範圍：(-5, 5)的整數
- 兩點要求：不能有相同的x坐標或y坐標
- 斜率要求：確保計算出的斜率為整數或簡單分數（分母不超過5）
- 截距要求：整數或簡單分數

難度5: 給定兩個整數坐標點，求通過這兩點的水平線或垂直線方程
- 坐標範圍：(-5, 5)的整數
- 兩點要求：必須有相同的x坐標（垂直線）或y坐標（水平線）
- 方程形式：垂直線時為 x = a，水平線時為 y = b（不需要寫成 y = 0x + b 的形式）

題型設計：
1. 根據難度生成適當的斜率(m)和y軸截距(c)
2. 對於難度4，生成兩點坐標，確保斜率為簡單分數
3. 對於難度5，專門生成垂直線或水平線，並使用最簡單的方程形式
4. 確保所有數值在合理範圍內
5. 分數答案需要化簡為最簡分數

錯誤選項設計：
1. 斜率正負號錯誤
2. 截距正負號錯誤
3. 斜率和截距互換
4. 分數未化簡
5. 計算過程中的常見錯誤（如兩點式計算錯誤）
6. 對於垂直線和水平線，混淆 x = a 和 y = b 的形式

題目呈現：
難度1,2,3：求通過點 P(x₁, y₁) 且斜率為 m 的直線方程（斜率截距式）

難度4,5：求通過以下兩點的直線方程：
A(x₁, y₁), B(x₂, y₂)

答案要求：
1. 方程必須寫成 y = mx + c 的形式（難度1-4）
2. 難度5的垂直線必須寫成 x = a 的形式，水平線必須寫成 y = b 的形式
3. 分數需要化簡為最簡分數
4. 係數1需要省略（如 y = x + 2，而不是 y = 1x + 2）
5. 係數-1只需寫負號（如 y = -x + 2，而不是 y = -1x + 2）

備註：
1. 確保所有數值在合理範圍內，避免計算過程出現過大或過小的數
2. 分數計算時需要約分至最簡
3. 特別注意正負號的處理
4. 注意分母不能為零（斜率不能為無限大）
5. 難度5的特殊情況需要使用最簡單的方程形式
